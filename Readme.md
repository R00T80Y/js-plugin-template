# How to use?
```
import PluginName from 'pluginName.plugin.js';
```

## Init
```
PluginName(document.getElementById('id'), {
  // Plugin options here
});
```

```
PluginName(document.querySelectorAll('.selector'), {
  // Plugin options here
});
```
